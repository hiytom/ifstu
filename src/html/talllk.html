if Talllk. 是 if Studio. 与其他社团、机构等共同举办的活动，活动形式主要有线下设计沙龙、聚会、演讲等，我们希望与设计师朋友以及热爱设计的朋友一起探讨设计，交流设计的想法，以及沟通更加美好的生活方式。
